SPIFFS File Server

This is a simple file server that serves files from the SPIFFS (SPI Flash File System) of the ESP8266. You can upload files to the SPIFFS using the Arduino IDE or any other tool that can write to the SPIFFS. The files will be available on the file server at the following URL: http://192.168.4.1/files

To view the files, simply navigate to the URL in your web browser. You can also download the files using the "Save As" option in your browser.

Note: The file server only serves files from the SPIFFS. It does not serve files from the SD card or any other storage device.
